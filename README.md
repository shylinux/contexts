# contexts

contexts通过集群化、模块化、自动化，可以在各种设备上，快速组建起云计算服务。

## install

### install by source
```sh
curl -fsSL https://shylinux.com |sh -s source
```

### install by binary
```sh
curl -fsSL https://shylinux.com |sh -s binary
```

## start

### serve
```sh
ish_miss_serve
ice>
```

### page
